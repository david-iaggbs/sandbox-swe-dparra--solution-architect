Scaffold a new product architecture design repo (arc42 + C4 + ADR + draw.io) or manage an existing one.

Arguments: $ARGUMENTS

Run the full scaffold-design skill workflow now.
- Use `scaffold <product-name>` to create a new design repo for a product.
- Use `new-adr <title> [--repo <product-name>]` to add an ADR to an existing repo.
- Use `update-section <03-12> [--adr <NNNN>] [--repo <product-name>]` to evolve architecture documentation.
